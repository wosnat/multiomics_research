## 2026-04-08 15:00 — Step 5: Metric design, verification, and scoring

### Sub-step 5a: Rank score formula design

**Formula:** For each signature gene in the target dataset:
1. concordance = +1 (same direction), -1 (opposite), 0 (not significant)
2. normalized_rank = 1 - (dir_rank / n_significant_in_direction) — normalizes within significant genes of the same direction
3. contribution = concordance × normalized_rank (0 for not-significant)
4. rank_score = mean(contribution) over matched genes (absent genes excluded)

**Why normalize by n_significant, not total_genes:** With total_genes (1849 for RNA-seq), all normalized ranks compress to ~0.95+. Rank 10 vs rank 200 barely differs. With n_significant (e.g., 405 sig_up), rank 10 → 0.975, rank 200 → 0.506. Meaningful spread.

This was discovered DURING the analysis — original implementation used total_genes, the researcher noticed rank_score ≈ hit_rate in the control separation plot, and we diagnosed and fixed the compression.

### Sub-step 5b: Toy test verification

31 tests across test_signature.py (17) and test_scoring.py (14). Hand-calculated expected values for:
- All concordant, mixed, absent gene cases
- Tie-breaking by best rank
- Discordant gene handling
- Symmetric extended classification
- Permutation test with <30 genes (returns NaN) and >30 genes (returns p-value)
- score_with_significance breakdown (n_matched, n_absent, n_concordant, n_reversed, n_not_significant, hit_rate)

All 31 tests pass after normalization fix.
